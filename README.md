# Financial-Dashboard 

## Financial Dashboard (Power BI)

A comprehensive Power BI dashboard designed to analyze financial performance, track key performance indicators (KPIs), and provide actionable insights into revenue, expenses, and profit margins.

---

## 🚀 Features & Insights

* **Executive Summary:** High-level overview of total revenue, net profit, and operational costs.
* **Trend Analysis:** Month-over-month (MoM) and Year-over-year (YoY) financial growth tracking.
* **Expense Breakdown:** Granular view of operational vs. fixed costs to identify saving opportunities.
* **Interactive Filters:** Dynamic slicing by region, product line, fiscal year, and quarter.

---

## 🛠️ Tech Stack & Skills Used

* **Tool:** Power BI Desktop
* **Data Modeling:** Star Schema (Fact and Dimension tables)
* **DAX (Data Analysis Expressions):** Used for advanced calculations, time-intelligence functions (YTD, MoM), and custom KPIs.
* **Power Query:** Data cleaning, transformation, and ETL processes.
